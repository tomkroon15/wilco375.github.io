Teleport Pad Mod

Explanation

This mod adds in a Teleport Pad, which can be crafted with a diamond and 8 redstone. When placed down it asks you to give it a name by typing the command /teleporter name <name>. Once you have placed down and named two or more teleporters, you can teleport between them by simply stepping on to one, and selecting the teleport pad you want to go to in the GUI that pops up.

A teleport pad should automatically be removed from the GUI once broken. If this isn't the case you can manually remove it with the command /teleporter remove <name>
