VoxelSniper Mod

Explanation

[Explanation]
This mod has two parts:
• VoxelSniping
You can spawn in structures by shooting arrows. Enable by clicking the "V" button and clicking the top button.
Options:
- You can spawn in Spheres, Hollow Spheres, Disks, Cubes, Hollow Cubes and Platforms
- You can set the replace mode to only replace blocks, air or both
- You can set any radius
- You can set the block

• Painting
This will replace any blocks in a set radius around you with the set block. Block and radius can be configured by tapping the "P" button
