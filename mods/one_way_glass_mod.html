One Way Glass Mod

Explanation

This mod adds a block that you can look through from one side, but not from the other. You can craft it with 7 iron ingots and 2 glass

If you want to change the iron block texture to a different block texture, type the command /glasstexture to change the texture for THAT WORLD (it will save the applied texture in the world directory, so if you'd share the world, it'd still have the same texture applied). After that, re-open the world. You can find a list of all the texture names here: http://zhuoweizhang.net/mcpetexturenames/
