Instant Structure Mod

Description

This is an early version of my Instant Structure Mod. It currently includes 67 structures, but (hopefully many) more will be added later.
If you have made a build that you'd like to have added to the mod, you can just post a download link below here, to a zip of your world with the build in it.
This mod has a built in update detection, that will let you know as soon as there's a new version availabe, so you can always have the newest builds!

How it works

You can find the needed items in the creative menu or cheat them in (id: 500 and 501). Click a block with the Structure Selector, and a GUI pops up, with a list of all the structures.
Select the one you want. After that, click on a block where you want to place that structure with the Structure Builder. It will popup with a warning dialog. If you click Yes, the structure will be placed.
Some structures will be placed in the direction you're facing, some will just be placed one direction. The warning dialog will let you know what's the case with that build.
