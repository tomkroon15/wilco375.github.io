Block Placer / Breaker Mod

Explanation

This mod adds 4 new blocks to the game:

Block Breaker
This block breaks any block in front of it and puts it into the chest behind it. It will always be active and can be placed in any orientation
Any blocks on this face of the block will be broken:

And will be put into the chest on this face of the block:


Redstone Block Breaker
Functions the same as the block breaker but only breaks a block when it receives a redstone signal


Block Placer
Functions the same as the block breaker but instead places any blocks that are in the chest


Redstone Block Placer
Functions the same as the block placer but only places a block on a redstone signal
