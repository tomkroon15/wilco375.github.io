Mo' Crafting Recipes Mod

Explanation

This mod adds in a bunch of new crafting recipes that allow you to make almost any block in Vanilla Minecraft!

[Show all recipes] [Hide all recipes]
8x Grass + 1x Dirt → 8x Podzol
8x Grass + 1x Brown Mushroom → 8x Mycelium
8x Dirt + 1x Long Grass → 8x Grass Block
1x Water Bucket → 1x Ice
1x Wheat Seeds → 1x Pumpkin Seeds → 1x Melon Seeds → 1x Beetroot Seeds → 1x Wheat Seeds
2x Paper + 1x Stone → 1x Bone
8x Paper + 1x Stone → 1x Egg
8x Stone + 1x Torch → 4x Glowstone
1x Water Bucket + 1x Gold Ingot → 1x Sponge
1x Gravel → 1x Flint
Convert any sapling into another one, for example, 1x Oak Sapling → 1x Spruce Sapling
1x Brown Mushroom → 1x Red Mushroom → 1x Brown Mushroom
8x Oak Leaves + 1x Diamond → 1x Emerald
1x Bone Meal → 1x Ink Sack (Because those Ink Sacks are far too rare...)
1x String → 1x Cobweb

You can also craft any Spawn Egg with 1x Diamond + 1x Egg and the following items:
Villager: 1x Emerald
Chicken: 1x Feather
Cow: 1x Raw Beef
Pig: 1x Raw Porkchop
Sheep: 1x Wool
Wolf: 1x Bone Meal (Because the Bone is for the Skellie :p)
Mooshroom: 1x Red Mushroom
Creeper: 1x Gun Powder
Enderman: 1x Gold (Why are there no Ender Pearls...)
Silverfish: 1x Gray Dye
Skeleton: 1x Bone
Slime: 1x Slimeball
Spider: 1x String
Zombie: 1x Feather + 1x Raw Beef (I still don't understand why Zombies drop feathers instead of Rotten Flesh...)
Zombie Pigman: 1x Feather + 1x Raw Porkchop
